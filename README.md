# Robotic Fleet Orchestrator

This project is an advanced warehouse automation system designed to manage and orchestrate a fleet of robots. It enables efficient task execution and real-time inventory management, exploring the core challenges of robotic fleet coordination in dynamic environments.

## Key Features
*   Centralized fleet management and task assignment for multiple robots.
*   Real-time inventory tracking and status monitoring.
*   Dynamic path planning and collision avoidance logic.
*   Simulation environment for testing coordination strategies.

## Tech Stack
*   Python
*   ROS (Robot Operating System)
*   Docker
*   PostgreSQL

## Getting Started
1.  Clone the repository: `git clone https://github.com/zoreanuj/robotic-fleet-orchestrator.git`
2.  Navigate to the project directory and install dependencies: `pip install -r requirements.txt`
3.  Set up the database: `docker-compose up -d`
4.  Run the core simulation: `python main.py`
